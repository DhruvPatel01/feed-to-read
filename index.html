    Version 2.0.0 RC2
    The second release candidate (RC2) for Nim version 2.0 is ready for testing. Don’t panic! One of our design goals was to make it easy to write code that works with Nim version 1 and 2. Many important packages already work with version 2 and as usual many innovations are behind switches that can be enabled or disabled on a per module level thanks to the .experimental pragma. Version 2 is based on the same codebase as version 1, it’s an evolution, not a revolution.
